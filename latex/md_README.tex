\subsubsection*{Dependencies}

The client needs Open\+S\+SL, Boost libraries and glog.

{\bfseries Mac OS X}

Open\+S\+SL 
\begin{DoxyCode}
brew install openssl
cd /usr/local/include
ln -s ../opt/openssl/include/openssl .
\end{DoxyCode}


Boost 
\begin{DoxyCode}
brew install boost
\end{DoxyCode}


Glog 
\begin{DoxyCode}
brew install glog
\end{DoxyCode}
