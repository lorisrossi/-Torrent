Parse a be_node dictionary corresponding to the "info" key of the torrent.

Parsed keys: name, piece length, pieces, length, files. Save all the information in a Torrent struct passed as a parameter. In the case of a single file torrent, this function parse file info without calling "parse_file_dict".
