Introduction

alphaTorrent is a CLI program that allow to download files by passing a .torrent file. For a detailed explaination of the application workflow see Application Workflow.

Installation

Step 1: Clone the repository

git clone https://github.com/lorisrossi/alphaTorrent.git

Step2: Install dependencies (Linux)

This command will install the openssl, boost and glog libraries

sudo apt-get install libssl-dev libboost-all-dev libgoogle-glog-dev

Step2: Install dependencies (Mac OS X)

This command will install the openssl

brew install openssl
cd /usr/local/include
ln -s ../opt/openssl/include/openssl .

Boost

brew install boost

Glog

brew install glog

Compiling

Go to the main folder and execute

make

Usage

./alphatorrent-cli torrent-file.torrent
