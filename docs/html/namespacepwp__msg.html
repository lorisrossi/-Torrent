Handler that is executed every KEEP_ALIVE_TIME and send a keep-alive message.

If the peer_c's socket is closed, then the periodic routine is stopped by calling.
timer.cancel() So the handler is calcelled from _io_service

Parameters
peer_cThe peer connection structure.
timerThe deadline_timer that call the handler.
