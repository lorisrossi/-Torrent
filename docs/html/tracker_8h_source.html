81  * port: The port number that the client is listening on.
82  * uploaded: The total amount uploaded in base ten ASCII.
83  * downloaded: The total amount downloaded in base ten ASCII
84  * left: The number of bytes this client still has to download in base ten ASCII.
85  * compact: Setting this to 1 indicates that the client accepts a compact response.
86  * no_peer_id: Indicates that the tracker can omit peer id field in peers dictionary.
